# MryLibrary

MryLibrary is communication library.

## Server Config

Server configuration in JSON format.

```
[
	{
		"name" : "gmsv",
		"host" : "localhost",
		"port" : 3000
	}
]	
```

- "name" : Used array key in program.
- "host" : Server host.
- "port" : Server port.
